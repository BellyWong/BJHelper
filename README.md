## 什么是BJHelper

BJHelper是一款完成iOS本地应用分发的示例，实现了现在iOS第三方应用市场本地安装app的功能。

## 如何安装

### 手动安装

```ruby
git clone https://github.com/beijiahiddink/BJHelper.git
open BJHelper
```

## 该怎样去使用

目前示例中的ipa包均属于个人签名，故出现无法安装的现象，请大家按照示例的样式自行更换需要安装的文件即可。
